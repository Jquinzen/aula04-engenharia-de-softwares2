##  Como rodar
1. Clonar o repositório:
```bash
git clone https://github.com/Jquinzen/aula04-engenharia-de-softwares2
```

2. Acessar a pasta:
```bash
cd repositorio
```

3. Instalar as dependências:
```bash
npm install
```
